Backend API sederhana berbasis Laravel menggunakan mock data JSON (tanpa database).

---

## Endpoint

| Method | Endpoint | Deskripsi |
|--------|----------|-----------|
| GET | `/api/items` | Tampilkan semua item |
| GET | `/api/items/{id}` | Tampilkan item by ID |
| POST | `/api/items` | Buat item baru |
| PUT | `/api/items/{id}` | Update seluruh data item |
| PATCH | `/api/items/{id}` | Update sebagian data item |
| DELETE | `/api/items/{id}` | Hapus item |
